# 📊 Customer Churn Analytics Dashboard

An interactive **Data Science dashboard** that predicts telecom customer churn using machine learning and explains the key drivers behind customer attrition.

The project combines **data analysis, machine learning, explainable AI, and an interactive dashboard** to provide insights into customer behavior and retention strategies.

# 📌 Project Overview

Customer churn is a critical business problem for telecom companies because acquiring new customers is significantly more expensive than retaining existing ones.

This project uses machine learning to:

• Predict whether a customer is likely to churn  
• Identify the most important factors influencing churn  
• Provide business insights through interactive visualizations

The project also includes a **Streamlit dashboard** for interactive analytics and predictions.

# 📈 Key Insights

From the churn analysis:

• Customers with **low tenure** have higher churn risk  
• **Fiber optic internet users** show increased churn probability  
• **Month-to-month contracts** significantly increase churn  
• Higher **monthly charges** correlate with customer attrition

These insights can help telecom companies improve **customer retention strategies**.

• Handles nonlinear relationships well  
• Works effectively with tabular data  
• Provides feature importance for explainability

# ▶️ Running the Project Locally

Clone the repository:


   git clone https://github.com/Shreya-devanapalli/customer-churn-analytics


Navigate to the project folder:


   cd customer-churn-analytics


Create virtual environment:


   python -m venv .venv


Activate environment:

   .venv\Scripts\activate


Install dependencies:


   pip install -r requirements.txt


Run the dashboard:


   streamlit run app.py
